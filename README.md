# 2023📊 갤럭시 S23 시리즈 데이터 분석을 통한 광고 방향성 제안
### 📝 프로젝트 소개
* **개발 환경:** Python, Streamlit
* **사용 라이브러리:** Pandas, NLTK, WordCloud, Matplotlib

---

### 💡 핵심 분석 결과 및 제안

**1. 기능 전략: '안정성' 강조**
* **분석:** 아이폰15는 '배터리/발열' 부정 언급이 많았으나, 갤럭시는 해당 이슈가 적음.
* **제안:** 강점인 카메라는 유지하되, 경쟁사 대비 우위인 **배터리 효율과 발열 제어 안정성**을 마케팅 포인트로 삼아야 함.

**2. 디자인 전략: '크림' 컬러 주력**
* **분석:** 판매량 분석 결과 **크림(1위) > 라벤더 > 그린 > 블랙** 순으로 선호도 확인.
* **제안:** 인기 색상인 **크림**을 고정으로 하고, 파스텔톤(라벤더)과 무채색(블랙/그린)을 조합한 라인업 구성 제안.

**3. 광고 전략: '일상 공감' 브랜딩**
* **분석:** 경쟁사(애플) 대비 독자적인 브랜드 충성도 이미지가 필요함.
* **제안:** 단순 스펙 나열보다는 **'일상', '공감', '희망'** 키워드를 활용한 스토리텔링 광고로 브랜드 팬덤 강화.
